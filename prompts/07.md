arefa 06: Corrigir Erro Unable to resolve module axios
Contexto: Após a nossa refatoração (na Tarefa 05), o seu services/ixcApi.ts foi modificado para chamar o seu backend (server.js). Para fazer esta chamada de API (via axios.create), o seu aplicativo (AplicativoFIber) precisa de ter a biblioteca axios instalada.

Análise do Erro: O erro Unable to resolve module axios from E:\Documentos\Github\AplicativoFIber\services\ixcApi.ts significa que:

O ficheiro ixcApi.ts (do seu aplicativo) tentou executar import axios from 'axios'.

O pnpm (gestor de pacotes) não conseguiu encontrar o axios na pasta node_modules do seu aplicativo (AplicativoFIber/).

Causa: Nós instalámos o axios no seu backend (dentro da pasta backend/), como mostra o package.json do backend. No entanto, nós não instalámos o axios no projeto principal (AplicativoFIber/), que é onde o ixcApi.ts está a ser executado.

Instruções de Correção
Para resolver este erro, você precisa de instalar o axios no seu projeto frontend (o aplicativo).

Abra o seu terminal.

Garanta que está na pasta raiz do seu aplicativo (NÃO na pasta backend/).

Bash

cd E:\Documentos\Github\AplicativoFIber
Execute o comando de instalação:

Bash

pnpm add axios
Verificação (O que vai acontecer)
Após rodar o pnpm add axios, verifique o seu ficheiro AplicativoFIber/package.json (o da raiz, não o do backend).

Você verá que o axios foi adicionado às "dependencies".

Depois disso, pode reiniciar o seu aplicativo (pnpm start) e o erro Unable to resolve module axios terá desaparecido.
